BusWatch: CTA

2014

C, PebbleSDK, JavaScript/JavaScript, HTML, CSS

BusWatch: CTA is a watch app I developed for the Pebble SmartWatch. It features the new Pebble 2.0 SDK JavaScript API to allow users to run the app without needing an Android or iOS companian app installed, making it simple and quick to use. In three easy steps, the user can keep an eye out for when the next bus is arriving at their chosen bus stop, receiving an alert when the bus is near.

The website is a quick one-page scrolling site built off of Twitter Bootstrap with analytics to track how far the user has scrolled and goals for if a user clicks the download link.

If you have a Pebble SmartWatch and live in Chicago, what are you waiting for? Take charge of the bus.
